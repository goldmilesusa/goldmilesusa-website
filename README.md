# Gold Miles USA — Premium Static Website

This project is intentionally simple: no Next.js, no npm, no build command.

## Cloudflare Pages deployment

1. Extract the ZIP.
2. Upload all files to the root of your GitHub repository.
3. In Cloudflare Pages:
   - Framework preset: **None**
   - Build command: leave empty
   - Build output directory: `/`
4. Deploy.

## Important items to edit before publishing

Open `index.html` and replace:

- `+1 (407) 555-0186` with the real company phone number.
- `contact@goldmilesusa.com` with the final business email if different.
- Any service descriptions you want to customize.
