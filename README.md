# Gold Miles USA — Website Version 2.0

This version removes the duplicated/baked-in hero text and uses a clean truck image with a single HTML text layer.

## Cloudflare Pages
- Framework preset: Next.js (Static HTML Export)
- Build command: `npm run build`
- Build output directory: `out`
- Root directory: leave blank

## Publish with GitHub Desktop
1. Open the local `goldmilesusa-website` repository.
2. Delete the old website files, but keep the `.git` folder.
3. Copy all files and folders from this package into the repository.
4. Commit as `Publish Gold Miles website version 2.0`.
5. Push origin.
